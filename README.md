# Gestión de Tareas

APlicación para gestionar tareas


## Notas:
Recuerden reconstruir las módulos de Node
```
npm install
```

Para construir el build, recurde:

* Productivo
La configuración esta dada por webpack.prod.js
```
npm run build
```
* Desarrollo
La configuración esta dada por webpack.configs.js
```
npm run build:dev
```
